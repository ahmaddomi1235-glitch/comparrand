# قارن دواءك

منصة صيدلانية رقمية لمقارنة الأدوية وعرض البدائل الدوائية في السوق الأردني.
الأسعار بالدينار الأردني (JOD).

**Owner:** RAND YAISH
**Developed by:** Ahmad Domi

---

## المزايا

- بحث الأدوية بالاسم أو المادة الفعالة
- مقارنة حتى 4 أدوية في جدول تفصيلي
- عرض البدائل الدوائية مع التحذيرات
- تحليل صورة الدواء بالذكاء الاصطناعي
- بحث ذكي عند عدم وجود نتائج محلية
- حفظ الأدوية المفضلة في المتصفح
- تجربة عربية كاملة (RTL)
- متوافق مع جميع الأجهزة
